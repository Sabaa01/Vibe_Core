# 🎧 VIBE_CORE

VIBE_CORE is a futuristic, cyberpunk-inspired music platform and audio lab built for artists, producers, and music lovers. It delivers an immersive digital experience where sound, design, and technology merge into one powerful interface.

## 🚀 Features

- Neon cyberpunk UI/UX design
- Interactive music track list with play/pause system
- Animated audio visualizers
- Responsive mobile-first layout
- Artist showcase grid with hover effects
- Modern glassmorphism design
- Smooth scrolling and UI animations

## 🎨 Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- CSS Animations

## 💡 Concept

VIBE_CORE is not just a website — it's a creative audio ecosystem. It simulates the feeling of a futuristic music lab where users can explore sound, discover creators, and experience music in a visually immersive way.

## 📱 Responsive Design

Fully optimized for:
- Mobile phones
- Tablets
- Desktop screens
